Pointers

A lot of this will likely be review if you're at all familiar with the concept of pointers but it's good to cover our bases anyways.

What is a pointer?

Simply put, a pointer is an object that stores the memory address of another object. This is useful for a lot of reasons:

  • It allows you to pass by reference instead of copying (duplicating) objects when you want to pass them to a function.
  • It allows you to change the value of a variable outside your current scope. As long as you have apointer to it, you can access the object's memory.
  • It allows you to store a collection of polymorphic objects as long as the type the collection stores is a class all objects inherit from.

There are more reasons to use pointers, but these are the main ones. Starting with the most basic form of a pointer in C++, the raw pointer, let's see how we might use one:


void addNumbers(int a, int b, int* c) {
    *c = a + b;
}

int main() {
    int result;
    addNumbers(4, 5, &result);
    std::cout << result << std::endl;
    return 0; 
}
